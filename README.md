# StayEase Grand Hotel - Final Frontend Project 🏨

StayEase is a comprehensive hotel booking web application developed as a final project for the Frontend Web Development track. The project is divided into two main parts: a responsive static website and an Angular-based Single Page Application (SPA).

## 📂 Project Structure

This repository contains:

### 1️⃣ Part One: Classic Web Interface (HTML, CSS, JS)
Built using semantic HTML, Bootstrap 5, and Vanilla JavaScript.
- **Home Page:** Features a hero banner, amenities, and footer.
- **Rooms Listing:** Dynamic rendering from JSON with search and filter functionality (Type & Price).
- **Booking System:** Form validation, date logic, and LocalStorage integration to save selections.
- **Dark Mode:** Persistent theme toggling using CSS variables and LocalStorage.

### 2️⃣ Part Two: Angular SPA (`my-hotel-booking` folder)
A modern Single Page Application built with **Angular 16**.
- **Components:** Navbar, RoomsList, RoomCard, and BookingForm.
- **Service & DI:** Centralized data management via `RoomsService`.
- **Directives:** Extensive use of `*ngFor`, `*ngIf`, and `ngSwitch`.
- **Communication:** Parent-child data flow using `@Input` and `@Output`.
- **Routing:** Seamless navigation between rooms and booking views.
